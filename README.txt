This is a Face Recognition Attendance System.

Run the .py file from the folder.
Choose User View to enter user data, to add days to the attendance sheet and to check the attendance excel sheet.

To add user data, enter Id and Name, and click on Take Images.
Images will be clicked for around 10 seconds and the screen will close.

THE PASSWORD FOR THE USER VIEW IS 1234

Once User data is input into the system, attendance can be taken from the User View.
Attendance will be confirmed in the Status.
Click on Check Attendance for the attendance percentage of the student who's attendance was recently taken.
